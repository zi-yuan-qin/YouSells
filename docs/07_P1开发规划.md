## 16. 进度管理

### 状态词汇

- 🔴 未开始
- 🔵 进行中
- ⛔ 阻塞中
- ✅ 已完成

### ⛔ 阻塞定义

任务在以下情况下标记为⛔ 阻塞中：
- 前置任务未完成
- 外部依赖（API、人员、决策）超过 24 小时无法推进

阻塞时必须立即通知队长，说明：阻塞原因 + 需要什么解除 + 可以先做什么。

### 每日更新格式

```
昨日完成：
今日计划：
阻塞（如有）：
PR / 分支链接：
测试结果：
```

### 完成定义

- 代码或产出完成
- 测试通过
- 文档或交接说明更新
- 审查者无需猜测即可验证

## 17. PR 规则

- 共享文件需队长审查
